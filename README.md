# Frantz Enterprise Agency

Full-service digital agency operations platform.

**19 agents · 14 skills · $30K MRR north star**

Built with React 19 + TypeScript + Vite 8 + Tailwind CSS v4 + Zustand + Recharts

---

## Live Preview

https://frantzenterprise.github.io/Frantz-Enterprise-Agency/

## Modules

- Dashboard with MRR stats and KPI tracking
- Client management with CRUD and health monitoring
- Sales pipeline with deal funnel visualization
- Project and task management
- Revenue tracking with charts
- Agency roster with org structure
- Skills inventory
- 7 service pages (Creative, Paid Media, SEO, Email, Social, Brand, Content)

## Development

```bash
npm install
npm run dev
```
